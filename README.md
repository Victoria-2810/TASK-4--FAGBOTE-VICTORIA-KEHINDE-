📊 Sales Performance Dashboard
Overview
This project is an interactive Sales Performance Dashboard built in Microsoft Excel to analyze and visualize sales data. The dashboard provides key insights into sales trends, customer activity, product performance, and order status, enabling data-driven decision-making.
Dashboard Preview
The dashboard presents key performance indicators (KPIs) and visualizations that help monitor business performance at a glance.
Key Metrics
Total Revenue: ₦1,264,761.96
Total Quantity Sold: 3,535 units
Total Customers: 1,200
Average Order Value (AOV): ₦1,053.97
Features
📈 Monthly Sales Trend
Tracks sales activity across the year to identify seasonal patterns and performance fluctuations.
📦 Quantity by Product
Displays the quantity sold for each product category, helping identify high-demand products.
🚚 Order Status Breakdown
Visualizes the distribution of order statuses:
Delivered
Shipped
Pending
Returned
Cancelled
💰 Product by Total Price
Shows revenue contribution by product category, enabling product profitability analysis.
📊 Product by Order Status
Analyzes how different products are distributed across various order statuses.
📅 Total Price by Year
Compares annual sales performance across multiple years.
